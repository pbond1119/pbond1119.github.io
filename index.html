MIDI Effects

Peter Bond

I/O:

Select your MIDI input device and MIDI output device. Your input will be the name of your MIDI controller, and your output will be a virtual MIDI port (IAC Driver Bus 1)

Transposer:

Using the slider, select the number of semitones that you want to transpose your MIDI signal by.

Chord Builder:

Using the dropdown menu, select which type of chord you would like to build off the note you are playing.
